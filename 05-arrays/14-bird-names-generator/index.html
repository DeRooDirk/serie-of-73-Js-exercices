5.14 Bird names generator

When you click the button display in the given tag a random name of bird with a random adjective based on the data provided in the script.
Each name is associated to a number that tells you how to decline it. If you see "1" you have to add "A" in front of the bird's adjective and name (example: "A tall robin"). If the number is higher than "1", you have to add "A few" in front of the name, and an "s" at the end (example: "A few perched pigeons").
